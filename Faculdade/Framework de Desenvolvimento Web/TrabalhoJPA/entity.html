EntityManager

O EntityManager é o serviço central para todas as ações de persistência, ele administra o O/R que o mapea entre classe de entidade e uma fonte de dados. Ele apenas se empenha naquilo que é necessário com as APIs, para criar as consultas, com isso ele busca objetos, sincronizando e inserindo no bando de dados, o mesmo é para os caching e pode administrar a interação entre uma entidade e serviços transacionais em um ambiente Java EE com JTA.

O EntityManager é integrado com Java EE e EJB, mas não é limitado a estes ambientes, pode ser usado em programas de Java SE também, por exemplo.

Para entender o Entity Manager é necessário entender o contexto de persistência, todo contexto de persistência é associado com uma unidade de persistência.

EntityManagers podem ser criados ou podem ser obtidos de um EntityManagerFactory. Em uma aplicação Java SE, você tem que usar um EntityManagerFactory para criar instancias de um EntityManager. Usar o factory não é uma exigência em um ambiente Java EE.
