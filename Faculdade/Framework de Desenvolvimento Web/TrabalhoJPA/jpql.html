JPQL

O Java Persistence query language (JPQL) é uma linguagem de consulta, ela pode ser definida em qualquer classe, dentro de um método por exemplo.

Usando o método createQuery(), podemos criar uma consulta passando uma String com o código JPQL. Est tipo de consulta é chamada de consultas dinâmicas.

Exemplo de consulta para buscar todas as entidades no banco:

Query consulta1 = em.createQuery("select a from Alunos a");
List alunos = consulta1.getResultList();
for(Alunos al: alunos){
System.out.println("Nome: " + al.getNome());
}

Foi apenas um exemplo, porém, existem várias funções interessantes que podem ser utilizadas para consultas, as mais comuns são: AVG, COUNT, MAX, MIN, SUM

Existem também as Named Queries, elas são definidas através de anotações dentro das classes que implementam a entidade. Para definir apenas uma consulta por exemplo, basta escrever a seguinte anotação: @NamedQuery

Para realizar várias consultas, basta escrever a seguinte anotação:@NamedQueries

Apenas por pesquisa existe os métodos que para executar uma Named Query, basta utilizar o método createNamedQuery().
